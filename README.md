# Proyecto VIDEOJUEGO -  3D en Unity

## Descripción

Videojuego desarrollado en Unity como proyecto académico orientado al aprendizaje de programación de videojuegos y simulación de físicas en entornos 3D.

El proyecto incluye control de personajes, movimiento de enemigos, interacción con objetos del escenario y mecánicas básicas de combate dentro de un entorno tridimensional.

## Características

* Movimiento de personajes mediante scripts en C#
* Sistema de físicas integrado con Unity
* Personajes controlados por el jugador
* Enemigos con comportamiento automático
* Escenarios 3D interactivos
* Gestión de animaciones y colisiones

## Tecnologías Utilizadas

* Unity Engine
* C#
* Visual Studio
* Física 3D
* Animaciones
* Desarrollo de Videojuegos

## Objetivo

Aplicar conceptos de programación orientada a objetos, diseño de videojuegos y simulación física para desarrollar una experiencia interactiva en un entorno 3D.
